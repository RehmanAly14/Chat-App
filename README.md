 # 💬 Chatty

### A modern real-time chat application built with the MERN stack, designed for fast, interactive, and seamless communication.

#### Chatty focuses on real-time performance, clean architecture, and user experience, making it a solid end-to-end full-stack project.

---

## Features

#### Instant Messaging
- Real-time conversations powered by WebSockets
#### Media Sharing
- Send and receive images effortlessly
#### Profile Customization
- Update profile pictures for a personalized experience
#### Online Status Tracking
- See who’s online and start chatting instantly
#### Global State Management
- Efficient state handling using Zustand
#### Modern UI/UX
- Clean and responsive interface with DaisyUI

---

## 🧩 Tech Stack
#### Frontend
- React.js
- Zustand
- Tailwind CSS
- DaisyUI
#### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
---

## 🔌 Real-Time Communication
- WebSockets (Socket.io)

---

## 📁 Project Structure
/client        → Frontend (React)
/server        → Backend (Express)
/models        → Database schemas
/controllers   → Business logic
/routes        → API endpoints
/sockets       → Real-time socket logic

---

### Getting Started
#### 1. Clone the repository
- git clone https://github.com/your-username/chatty.git
- cd chatty
#### 2. Install dependencies
- Frontend
- cd client
- npm install
#### Backend
- cd server
- npm install

---

### Environment Variables
#### Create a .env file inside the server folder:

- MONGO_URI=your_mongodb_uri
- JWT_SECRET=your_secret_key
- PORT=5000
---

### Run the application
#### Backend
- npm run dev
#### Frontend
- npm run dev
